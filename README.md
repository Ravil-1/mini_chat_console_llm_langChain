# Mini Chat Console LLM

Это просто тестовый консольный чат-бот, созданный для экспериментов с языковыми моделями (LLM) и библиотекой LangChain.

## Описание

Проект представляет собой простое консольное приложение, реализующее чат-бота. Используется для тестирования базового взаимодействия с LLM.

## Использование

1. Убедитесь, что у вас установлены все зависимости из `requirements.txt`.
2. Запустите скрипт `main.py`.

```bash
python main.py
```

## Зависимости

- Python 3.x
- langchain
- langchain_ollama

## Настройка LLM

В файле `main.py` необходимо указать правильный URL для Ollama сервера. В текущем коде используется:

```python
base_url="http://192.168.1.104:11434"
```

**Замените этот адрес на актуальный IP-адрес и порт вашего Ollama сервера.**

Также можно изменить модель, указав другое значение параметра `model`.
