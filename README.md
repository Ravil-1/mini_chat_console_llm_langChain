# Mini Chat Console LLM

Это просто тестовый консольный чат-бот, созданный для экспериментов с языковыми моделями (LLM) и библиотекой LangChain.

## Описание

Проект представляет собой простое консольное приложение, реализующее чат-бота. Используется для тестирования базового взаимодействия с LLM.

## Использование

1. Убедитесь, что у вас установлены все зависимости.
2. Запустите скрипт `main.py`.

```bash
python main.py
```

## Зависимости

- Python 3.x
- langchain
